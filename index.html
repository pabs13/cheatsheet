HTML Table Tags

Name Tag Description
Table <table> The wrapper element for all HTML table elements.
Table Head <thead> The set of rows defining the column headers in a table.
Table Body <tbody> The set of rows containing actual table data.
Table Row <tr> The table row container.
Table Data <td> The table data container.
Table Header Data <th> The table header data container for header or footers.
Table Footer <tfoot> The set of rows defining the footer in a table.

HTML Attributes for Tables

Name Tag Description
Column Span colspan Defines how many columns a td element should stretch across.
Row Span rowspan Defines how many rows a td element should stretch through.
